Portfolio site
=============================

Установка
------------

Клонируйте репозиторий или скачайте его архивом. Выполните команду "npm i" в терминале.

Структура
------------

      fonts/               шрифты
      icons/               иконки
      images/              картинки
      js/                  js файлы
      style/               файлы стилей (scss)
      template/            разметка (jade)
